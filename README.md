## Сборка и запуск

### Требования

- Qt 6.5.2 (модули: Core, Network, Widgets)
- CMake 3.16+
- Компилятор с поддержкой C++17 (MSVC 2019+, GCC 9+, Clang 10+)
- Qt Creator (опционально, для удобной сборки)

### Сборка через Qt Creator

1. Открыть корневой `CMakeLists.txt` в Qt Creator.
2. Настроить комплект (Kit) с Qt 6.5.2.
3. Выбрать цель сборки: `QtTelecomServer` или `QtTelecomClient`.
4. Собрать проект (Ctrl+B / Cmd+B).
5. Запустить (Ctrl+R / Cmd+R).

## Запуск

### 1. Запуск сервера

В директории `build` уже находятся предварительно скомпилированные исполняемые файлы для Windows:

```bash
./QtTelecomServer.exe
```

В открывшемся окне:
1. Нажмите **«Запустить сервер»** — сервер начнёт слушать порт 12345.
2. Запустите один или несколько клиентов (см. ниже).
3. В таблице клиентов появятся записи с ID, IP и статусом.
4. Нажмите **«Старт клиентов»** — клиенты начнут отправку данных.
5. Данные отобразятся в таблице «Данные от клиентов».
6. Через **«Настройки»** можно сконфигурировать пороговые значения.

### 2. Запуск клиента (по умолчанию)

```bash
./QtTelecomClient.exe
```

Подключение к `127.0.0.1:12345`.

### 3. Запуск клиента с параметрами

```bash
./QtTelecomClient.exe 192.168.1.100 12345
```
